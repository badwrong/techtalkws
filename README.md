# TechTalk Webservice

This is a starter project for the backend portion of the TechTalk pairing interview excercise.

## Tech Stack
    - Gradle
    - Spring Boot
    - H2 in-memory DB
    - JPA
    - Spock (for automated tests)
    
## Accessing the H2 Console
http://localhost:8080/h2-console
